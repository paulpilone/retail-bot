## Dependencies and Setup

### Prerequisites

- [nvm](https://github.com/nvm-sh/nvm) and Node v23.13.1

### Install Node

```sh
nvm install
nvm use
```

### Install Dependencies

```sh
npm install
```

## Tests

```sh
npm run test
```

## Usage

```sh
npm run bot
```
